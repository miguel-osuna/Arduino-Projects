# Arduino-Projects
This repository contains the code of different Arduino projects that were made with electronic components such as:

- Push Button
- Piezoelectric Buzzer
- DC Motor
- Potentiometer
- 7 Segment Display
- 4 digit 7 segment display
- LEDs
- Shift Register
- Servomotor
- Joystick
- Ultrasonic Sensor
- H-Bridge
- LED Matrix
